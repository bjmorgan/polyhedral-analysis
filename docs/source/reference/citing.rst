Citing
======

If you use polyhedral-analysis in your research, please cite the
repository:

    B. J. Morgan, *polyhedral-analysis*,
    https://github.com/bjmorgan/polyhedral-analysis

The continuous symmetry measure methodology is described in:

    M. Pinsky and D. Avnir, "Continuous Symmetry Measures. 5. The
    Classical Polyhedra", *Inorganic Chemistry*, **37**, 5575--5582
    (1998). https://doi.org/10.1021/ic980060v
